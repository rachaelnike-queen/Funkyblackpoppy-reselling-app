# FBP Reselling App — Stable Single-App v9

This is the stabilized single-file app build for Funky Black Poppy reselling workflows.

## Open the app

Unzip the project and double-click `index.html`.

## Main tools

- Dashboard
- Listing Wizard
- Style Cards
- Photo Studio / Image Generator

These tools are embedded as internal views inside `index.html`, so the app no longer depends on separate page links for the main workflow.

## GitHub upload

Upload the contents of this folder to your GitHub repository with `index.html` at the repo root.
